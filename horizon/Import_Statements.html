2.1.2 Import Statements

Import statements come in a few forms:

import-stmt: IMPORT import-source AS NAME

import-stmt: IMPORT NAME (COMMA NAME)* FROM import-source

import-source: import-special | import-name | import-string

import-special: NAME PARENNOSPACE STRING (COMMA STRING)* RPAREN

import-name: NAME

import-string: STRING

The form with

import-name

looks for a file with that name in the built-in libraries of Pyret, and it is an error if there is no such library.

Example:

import equality as EQ check: f = lam(): "" end EQ.equal-always3(f, f) is EQ.Unknown end
